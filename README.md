# Projecte Bloc 6 - Desenvolupament Web amb Django

## 1. Introducció
Aquest projecte consisteix en el desenvolupament d'una aplicació web de tipus **Blog** utilitzant el framework **Django**. L'objectiu principal és posar en pràctica la creació de models de dades, la gestió de rutes, la creació de vistes dinàmiques i l'administració de continguts mitjançant el panell de gestió propi de Django, així com la integració de sistemes de desplegament i documentació contínua.
